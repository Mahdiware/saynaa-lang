#### Contributing to Saynaa
Everyone is welcome to participate in Saynaa. Contribution encompasses more than just submitting pull requests;
there are numerous ways for you to engage, including reporting issues, introducing new features,
or actively engaging in the project.

#### Where to implement alterations
Core libraries and documentation are likely the areas that require significant improvement.
Any modification to the core VM or other segments of the compiler is highly encouraged.

#### Testing recommendations
If you resolve a bug or introduce new functionality, When you are done run the python script "util/test.py" and "util/check.py"

#### Adding Documentation
When adding new functionality or modifying the existing functionality,
ensure you include or edit the documentation to reflect these changes so that there is documentation available for new users.

#### Contributors File
Remember to include your name and github handle and email address in the official **CONTRIBUTORS** file!
